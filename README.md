## 功能

- 使用者可以註冊、登入
- 使用者可以購買鮮食、加入購物車
- 使用者可以建立訂單、並直接查看店家地圖
- 使用者可以查看已成立的訂單歷史資料

---

## Built With

- **template of:** vite + react
- **tailwindcss:** v3.x

### Started

```bash
yarn
# or `npm install`
```

---

### Develop

```bash
yarn dev
# or `npm run dev`
```

---

### Build

```bash
yarn build
# or `npm run build`
```
